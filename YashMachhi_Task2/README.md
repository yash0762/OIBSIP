## 📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) to analyze the sharp and sudden increase in the unemployment rate across different states in India during the COVID-19 pandemic in 2020.

## 📂 Folder Files
* `Unemployment in India.csv` - Data containing Rural vs Urban tracking splits.
* `Unemployment_Rate_upto_11_2020.csv` - Clean monthly state-wise tracking data.
* `YashMachhi_Task2.ipynb` - Python data processing and visualization code.
* `unemployment_timeline_spike.png` - Graph tracking the lockdown timeline shock.
* `regional_severity_ranking.png` - Bar chart ranking the hardest-hit states.
* `rural_vs_urban_impact.png` - Boxplot highlighting geographic sector differences.

## 📊 Core Insights Detected
1. **The Lockdown Shock:** Line charts show an unprecedented vertical spike in national unemployment immediately following the March 2020 strict lockdown phase.
2. **State-Level Disparity:** Certain states faced much harsher economic disruptions compared to others due to differences in regional industry structures.
3. **Urban vs Rural Stress:** Boxplots show that urban job markets generally faced wider fluctuations and higher peak stress compared to rural agricultural sectors.

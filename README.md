## 📂 Repository Portfolio Structure

### 🌸 [Task 1: Iris Flower Classification](./YashMachhi_Task1/)
* **Objective:** Train a K-Nearest Neighbors (KNN) Machine Learning model to accurately classify iris flowers into three distinct species based on morphometric features.
* **Tech Stack:** Python, Jupyter Notebook, Pandas, Scikit-Learn, Seaborn.
* **Key Outcome:** Achieved near-perfect classification accuracy scores across validation splits.

### 📉 [Task 2: Unemployment Analysis with Python](./YashMachhi_Task2/)
* **Objective:** Conduct an extensive Exploratory Data Analysis (EDA) on macro-economic timeline shocks and regional labor force vulnerabilities in India during the 2020 COVID-19 pandemic lockdowns.
* **Tech Stack:** Python, Jupyter Notebook, Pandas, Matplotlib, Seaborn.
* **Key Outcome:** Isolated strict lockdown timelines, state-by-state severity trends, and urban vs. rural resilience factors.

### 🚗 [Task 3: Car Price Prediction with Machine Learning](./YashMachhi_Task3/)
* **Objective:** Construct a data-driven Supervised Regression model pipeline to estimate the numerical valuation of used passenger vehicles.
* **Tech Stack:** Python, Scikit-Learn pipelines, ColumnTransformer, OneHotEncoder, Linear Regression.
* **Key Outcome:** Created an integrated vector transformation workflow to translate categorical attributes into continuous metric forecasts.

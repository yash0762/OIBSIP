## 📌 Project Overview
The objective of this project is to train an end-to-end Machine Learning model capable of learning from the morphometric measurements of Iris flowers (Sepal Length, Sepal Width, Petal Length, and Petal Width) to accurately classify them into their respective species: *Setosa*, *Versicolor*, and *Virginica*.
